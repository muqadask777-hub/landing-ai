# Assignment No. 3: Landing Page Development (Version A - Manual)

### Description
This project is a completely manual implementation of the clothing store landing page layout. Every line of HTML and CSS has been written from scratch without utilizing templates or AI code generators to master core styling concepts.

### Concepts Learned & Applied:
- Custom layouts using standard CSS **Floats** and **Inline-Blocks**.
- Managing percentages (`width: 50%`, `width: 23%`) for basic multi-column structure grids.
- Box model styling including strict custom padding and margins.
